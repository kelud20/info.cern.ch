Mail Robot

The mail robot is a program which will accept incoming mail and allow remote users to:
  • Subscribe to mailing lists (and unsubscrite)
  • retrieve information given a W3 addresss (URL)
Originally from UC Berkeley, an enhaned robot is distributed as part of the world-wide web global information initiative. Further information available is:
